# Install

## Linux based OS install:
```
git clone https://github.com/sign5/apk-differ && cd ./apk-differ
pip3 install -r requirements.txt
```

# Usage example:

```
python3 main.py <path-to-old-apk-file> <path-to-new-apk-file>
```
